Sobre o Sisfrota
Sistema web desenvolvido para controle e gestão de frotas de veículos. Permite o cadastro de marcas, modelos e veículos, com rastreamento de horímetro e ano de fabricação. Dados persistidos via API Xano com arquitetura separada em serviços e controllers. Projeto desenvolvido no SENAI.
